As Mistral grew, we realised that aspiring developers lacked relevant essential skills. The aim of Gigi School of Coding is to change that. This program started back in March 2015, and to date, 57 amazing people went through the program; 35 got a job at Mistral and the rest in other successful companies around the world. During this program, you will “live” Mistral for 4 months/ 8 hours a day, get to know our employees and join in on all epic Mistral activities.
Being part of the program, you get a chance to become a junior at Mistral and to contribute in making the web a better place for the biggest clients alongside the finest teams. Upon successful completion of the program, we expect you to be able to get the requirements from a client, analyze them, make a project plan, design a database, draw wireframes, make mockups, write code to provide Web API services and finally convert data to useful information directly to a user’s screen.

Courses

MongoDB & Mongoose
MongoDB is a very popular NoSQL database that is fast and scalable. This course will introduce you to this unique database and arm you with the essential skills to start using Mongo to power your software application.
NodeJS
The Node.js runtime powers back-end servers for big players like PayPal, Netflix, LinkedIn, and even NASA. This course will teach you the fundamentals of this very popular runtime and get you comfortable writing code for Node.
Express
Learn how to build a Node.js application that can perform create, read, update and delete (CRUD) operations. Learn how to build a RESTful services with Node.js and Express, manipulating data in MongoDB and consuming services.
React
React is one of the world's most popular libraries for creating web user interfaces. Learn how to confidently construct React applications that are simple and easy to maintain. Learn how to use Redux, React Router, and ES6 to build a real world app with React.
Material Design
Learn how to customize Material Design to change the look and feel of your UI, expressing brand and style through elements like color, shape, typography, and iconography.
React Native
React Native is quickly becoming the preferred hybrid mobile app platform. This course will teach you the essentials of React Native including creating components, using props and state, and making use of data from remote sources.
